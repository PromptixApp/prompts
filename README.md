# Base prompts for Promptix

This directory contains the core prompt configurations and templates used by the Promptix application.

## Files Description

- **`prompts.json`**: A JSON collection of predefined prompts available in the application (e.g., Translate, Summarize, Improve Writing). Each entry includes the title, user prompt template, system prompt, and metadata.
- **`system.txt`**: The text file containing the system prompt for the "Promptix Assistant". This assistant helps users generate custom prompts by understanding their requirements and producing structured prompt outputs.
